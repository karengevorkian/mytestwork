Три команды для экспортирования файлов и их изменений на рабочем компе:

git add . //точка означает, добавить все что в это папке. если хотим конкретный файл добавить, тогда вместо точки пишем название файла и его формат
git commit -m 'название комментария в репазитории'
git push

Работа с ветками в Main:

git branch // узнать текущую ветку 
git branch develop //создание ветки, ее название записывается после "branch" (commit и push команды делать?)
git checkout -b develop // смена ветки на develop
git push origin develop // добавить файл в конкретную ветку. сначала шаги создания/изменения файла и на шаге push, после него пишем origin и ветку

Сменить комментарий в файле. зайти в commet в git. там во вкладке Commits найти нужно и скопировать id
